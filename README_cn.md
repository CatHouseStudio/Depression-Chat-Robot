## 概述

### 项目目的

```
本项目旨在提供社交媒体内的聊天机器人，用于对抑郁症等负面心理疾病的防治与环节。

当然，本项目不能替代药物等常规治疗方式。
```

### 项目背景

```
项目起因是现阶段精神科医生与精神卫生中心医生的作为存在缺失，就诊过程充斥着开药，调整药物，缺乏必要的心理疏导与沟通。
```
